Why we use the Normalize CSS Script

Different browsers have different default styles, making HTML display differently on different browers. So, we use the normalize CSS script because it applies its own default styles that override the browser default settings, and therefore makes the webpage display the same on all browsers. This is useful is maintaining cross-browser consistency.

What is Semantic Markup

Semantic markup is a way of organizing and structuring content in a way that makes sense semantically, making it more readable. It ensures that the organization of the content reflects the meaning and type of content. By using semantic markup, the tags in HTML convey meaning about the content in addition to affecting how it is interpreted and used for styling. It is also useful for search engine optimization.
